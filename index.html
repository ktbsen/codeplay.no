About

This is a simple project overview page demonstrating the use of CSS variables for all styling settings.

Features

  • Clean and minimal design
  • Fully customizable through CSS variables
  • Responsive layout
  • Easy to maintain and extend

Getting Started

Edit the CSS variables in the :root selector to customize the look and feel of this page.

Available Variables

  • Colors: bg-color, text-color, heading-color, accent-color
  • Typography: font-family, font-size-*, line-height
  • Spacing: spacing-xs through spacing-xl
  • Layout: border-radius, max-width
